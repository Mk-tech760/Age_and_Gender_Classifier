# Age and Gender Classification using Convolutional Neural Networks

## 📌 Introduction
Age and gender are fundamental attributes in social interactions. Automating their estimation from facial images has been challenging, especially under unconstrained conditions. This project leverages **Convolutional Neural Networks (CNNs)** to classify age and gender more accurately, tested on the **Adience benchmark dataset**.

Our approach demonstrates that even with limited labeled data, a simple CNN architecture can outperform traditional methods and set a strong baseline for future improvements.

---

## 🎯 Objectives
- Develop a CNN-based framework for age and gender classification.
- Achieve higher accuracy compared to existing methods.
- Provide a simple, efficient architecture that can be extended with larger datasets.

---

## 📚 Literature Survey (Highlights)
- **Local Binary Patterns (LBP):** Texture-based facial descriptors for recognition.
- **AdaBoost for Sex Identification:** Fast and efficient classifiers achieving >93% accuracy.
- **Metric Learning (RCA):** Improves clustering and dimensionality reduction.
- **Label-Sensitive Learning:** Exploits ordinal relationships in age estimation.
- **Gabor Features + Fuzzy LDA:** Enhances coarse age classification in consumer images.
